# 多智能体协作工作流

预定义的多智能体协作模板，定义了参与的智能体、协作顺序和每步 prompt。

## 使用方式

每个工作流文件包含：
1. **参与智能体** — 需要激活的专家
2. **协作流程** — 每步做什么、谁来做
3. **Prompt 模板** — 直接复制粘贴使用

按顺序激活智能体，将上一步的输出作为下一步的输入。
